# thinkpad-savebatteryryzenadj
a simple script using ryzenadj for using custom tdp and save battery  with a menu using wofi-rofi, for hyprland systems

i created this menu just to improve my thinkpad battery with ussing the less ammount of battery when i need it, my thinkpad model is a T14 G2(amd), but you can try it on any device(amd) and change the settings of the tdp and add differents settings based on your needs.
